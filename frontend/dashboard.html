Automatic updates

Keep this box on the latest ShellTeam release. When on, a nightly check fast-forwards to the newest release tag and restarts the services, rolling back automatically if the update comes up unhealthy. Local code changes are never touched — a dirty or diverged checkout skips the update and reports it below.

Important — please read carefully

ShellTeam's AI agents take real actions on this machine: they run commands, install packages, manage services, and edit files — with no container between them and the box.

  • Agents may produce unexpected results or take unintended actions.
  • You are responsible for what agents do on your box.
  • Connecting third-party apps grants agents access to those services.
  • Treat the box as dedicated and recoverable — keep anything irreplaceable backed up elsewhere (see SECURITY.md).

ShellTeam is provided "as is", without warranty of any kind — you use it at your own risk.
